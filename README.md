  <h1>2048 with tilt controls</h1>
</div>

Recreated the 2014 game on a Tiva C Series Launchpad, displayed on a 64x64 LED Matrix. The game is controlled by tilting the screen around to move the tiles left, right, up, and down.



# Features:
- Block animations for movement and spawning
- Tilt controls
